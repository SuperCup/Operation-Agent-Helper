# 即时零售运营AI助手

一个专为到家平台活动运营与广告投放设计的AI助手系统，帮助运营人员高效完成从方案设计到效果评估的全流程工作。

## 功能特性

### 🎯 核心功能

1. **项目全生命周期管理**
   - 项目准备阶段：AI生成运营方案
   - 项目启动阶段：自动输出执行计划
   - 项目执行阶段：生成提报表并模拟平台提交
   - 效果监控阶段：实时数据收集与评估
   - 项目结束阶段：自动生成结案报告

2. **智能工作流引擎**
   - 可视化的AI执行步骤展示
   - 人机协作机制，关键节点需人工确认
   - 实时执行日志与进度追踪
   - 异常处理与人工介入提醒

3. **知识库系统**
   - 优秀运营案例沉淀
   - 历史投放数据分析
   - 智能案例推荐
   - ROI性能指标展示

4. **数据分析看板**
   - 多维度数据可视化
   - GMV、ROI、转化率等核心指标
   - 趋势分析与同比对比
   - AI数据洞察建议

5. **文档管理中心**
   - AI自动生成运营文档
   - 多版本管理
   - 在线预览与编辑
   - 文档导出功能

### 🎨 UI/UX 特色

- **高级感设计**：现代化的渐变色彩、流畅的动画过渡
- **直观交互**：清晰的信息层级、便捷的操作流程
- **响应式布局**：完美适配桌面端和移动端
- **状态反馈**：实时的加载状态、明确的操作提示

## 技术栈

- **前端框架**：React 18 + TypeScript
- **状态管理**：Zustand
- **路由**：React Router v6
- **样式**：Tailwind CSS
- **图表**：Recharts
- **图标**：Lucide React
- **构建工具**：Vite
- **日期处理**：date-fns

## 核心流程说明

### 1. 创建项目

1. 点击"创建项目"按钮
2. 填写项目基本信息（名称、品牌、平台、预算等）
3. 系统自动将项目设置为"准备阶段"

### 2. 启动AI工作流

1. 进入项目详情页
2. 点击"启动AI工作流"
3. AI开始执行当前阶段的工作流
4. 实时查看执行步骤和日志

### 3. 人工介入确认

- 当AI遇到需要人工决策的节点时会暂停
- 系统展示具体的问题和建议
- 运营人员确认后继续执行

### 4. 文档生成与查看

- AI自动生成各阶段所需文档
- 在文档中心查看所有文档
- 支持预览、下载、编辑

### 5. 数据监控与优化

- 实时查看运营数据
- AI提供优化建议
- 根据数据调整策略

## 模拟数据说明

项目包含完整的模拟数据，包括：

- **3个示例项目**：涵盖不同阶段和状态
- **工作流数据**：包含完整的执行步骤
- **文档数据**：运营方案、执行计划等
- **知识库**：3个优秀案例
- **数据指标**：7天的运营数据

所有数据都在 `src/data/mockData.ts` 中定义，可根据需要修改。
